# Icon Generation Instructions

This directory should contain application icons in various formats.

## Required Icons

| File | Platform | Size | Format |
|------|----------|------|--------|
| `icon.png` | All | 512x512 | PNG |
| `icon.ico` | Windows | Multi-size | ICO (16, 32, 48, 64, 128, 256) |
| `icon.icns` | macOS | Multi-size | ICNS |
| `tray-icon.png` | Windows/Linux | 16x16, 32x32 | PNG |
| `tray-iconTemplate.png` | macOS | 16x16, 32x32 | PNG (black only, template) |
| `icons/` | Linux | Various | PNG directory |
